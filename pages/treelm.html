Treelm

Tree-based Elemental Mesh

TreElM is the core library of APES providing the mechanisms to work with the octree mesh. It enables the implementation of stencil-based algorithms on the mesh by taking care of the neighborhood identification in a parallel environment.

Its public source code repository is hosted on github. Please see the documentation for its usage and more details.
