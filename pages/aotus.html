Aotus

Advanced Options and Tables in Universal Scripting

The Aotus library provides a Fortran wrapper to the Lua [extern] C-API and various convenience functions to ease the use of Lua scripts for the definition of parameters in Fortran applications. This part is completely independent of the other tools in APES and can easily be used on its own.

Its public source code repository is hosted on github. Please see the documentation for its usage and more details.

Aotus is the Night Monkey [extern]. For APES it talks to the moon (Lua). All tools of APES use Aotus to read configuration files.
